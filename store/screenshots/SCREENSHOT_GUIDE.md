# RoutineFlow Screenshot Guide

## Required Screenshots

### Screen 1: Home (루틴 탭)
- Show morning routines with some completed, some pending
- Bloom companion visible at top
- Time slot sections visible (Morning/Afternoon/Evening)
- Caption: "하루 루틴을 한눈에 관리하세요"

### Screen 2: Bloom Companion
- Show Bloom at Stage 3-4 with healthy animation
- Water drops and growth stats visible
- Caption: "루틴과 함께 성장하는 Bloom"

### Screen 3: AI Coach Chat
- Show conversation with Coach Lumi
- Include motivational message and routine advice
- Quick suggestion chips visible
- Caption: "AI 코치 루미의 맞춤 조언"

### Screen 4: Analytics Dashboard
- Show weekly completion chart
- Category breakdown pie chart
- Streak information
- Caption: "스마트 분석으로 습관을 돌아봐요"

### Screen 5: Community & Challenges
- Show active challenges with progress
- Community posts with likes
- Caption: "함께하면 더 쉬운 습관 만들기"

### Screen 6 (Optional): Widget Preview
- Show iOS/Android home screen widget
- Today's routines overview
- Caption: "위젯으로 빠르게 확인"

| Type | Minimum Resolution |
|------|-------------------|
| Phone | 1080 x 1920 |
| Feature Graphic | 1024 x 500 |

## Tips
- Use actual app data (not empty states)
- Show Korean language interface for Korean store
- Show English interface for international store
- Keep device frame consistent
- Use Figma or screenshots.pro for device mockups
- Dark mode screenshots are optional but recommended as secondary set
